# Services proposés

## Création de sites web

### Description
- **Type de sites :** Site vitrine / landing page
- **Inclus :**
  - Structure claire (présentation, services, contact)
  - Design moderne et responsive
  - Intégration du contenu fourni
  - Formulaire de contact
  - Déploiement (VPS ou plateforme cloud)
- **Délais habituels :**
  - 48 à 72 heures pour un site simple

### Cas d’usage typique
- Présenter une activité professionnelle ou associative
- Disposer d’un site simple et clair pour être contacté
- Mettre en ligne rapidement un projet ou une idée

## Chatbots IA avec base de connaissance

### Description
- Chatbot entraîné sur des **documents fournis** (pages, FAQ, PDF, notes)
- **Réponses contrôlées** : le bot répond uniquement à partir de la base de connaissance
- Pas d’hallucination ou d’invention d’informations
- Recherche sémantique pour répondre aux questions
- Déploiement inclus et intégration sur site web

### Cas d’usage typique
- Répondre automatiquement aux questions fréquentes
- Présenter des services ou informations 24/7
- Aider les visiteurs à trouver rapidement une information
- Réduire le temps passé à répondre aux mêmes questions

## Aide / accompagnement technique

### Description
- Aide ponctuelle ou récurrente
- Debug de code
- Aide à l’architecture logicielle
- Déploiement et infrastructure (Docker, VPS, Nginx)
- Accompagnement sur des sujets liés à l’IA

### Ce que je ne propose pas
- Support technique 24/7
- Développement d’applications mobiles natives complexes
- Gestion de campagnes marketing ou SEO avancé
- Projets sans besoin clairement défini
