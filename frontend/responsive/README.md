# Responsive Layer

This directory contains the frontend-specific responsive layer that sits on top of the main Tailwind-driven UI.

Current file:

- `mobile.css`: small-screen overrides used by the portfolio for button density, hero title scaling, card heading scaling, and stacked mobile form actions

Usage:

- imported from `frontend/src/index.css`
- intended for responsive-only overrides that should stay separate from the main design system styles
